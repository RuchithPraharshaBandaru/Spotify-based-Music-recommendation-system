# Spotify Music Recommendation Engine 

A machine learning based music recommendation system developed using Spotify kaggle dataset. The project analyzes audio characteristics and listening patterns to generate personalized song recommendations.

---

##  Project Overview

This project focuses on building a recommendation engine capable of suggesting songs based on user preferences and track similarity. By leveraging Spotify audio features such as danceability, energy, acousticness, loudness, tempo, valence, and speechiness, the system identifies patterns between tracks and recommends relevant music to users.

The recommendation pipeline combines data analysis and machine learning techniques to improve personalization and music discovery.

---
